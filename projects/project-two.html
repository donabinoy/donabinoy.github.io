AssetsADM

Project Overview

AssetsADM is an enterprise-level asset tracking application designed to improve operational efficiency for businesses. The app digitizes the traditionally manual process of asset management, providing a robust system for tracking company assets, maintaining audit trails, and managing inventory. I was responsible for building and publishing this application.

Key Features

  • Asset Tagging: Ability to add new assets by scanning barcodes or QR codes.
  • Real-Time Tracking: Track the location and status of each asset within the organization.
  • Audit Trails: Maintained a secure and immutable log of all asset movements and changes for accountability.
  • Reporting: Generated reports on asset utilization, maintenance schedules, and inventory levels.

Technologies Used

The app was developed natively using Swift and UIKit, following the MVC (Model-View-Controller) architecture. I integrated the device's camera for barcode scanning functionalities and ensured that all data was securely stored and synchronized with the central database via RESTful APIs.
