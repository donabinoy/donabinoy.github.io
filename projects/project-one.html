Sunrise Connect App

Project Overview

Sunrise Connect is a comprehensive healthcare application designed to bridge the gap between patients and doctors. I led the development of this application, which allows users to seamlessly schedule doctor appointments and conduct secure, high-quality video consultations directly from their mobile devices. The application was successfully built and published on the Apple App Store.

Key Features

  • Appointment Scheduling: Intuitive interface for users to find doctors and book appointments based on availability.
  • Secure Video Consultations: Integrated a HIPAA-compliant video conferencing feature to ensure patient privacy and data security.
  • User Profiles: Separate, secure profiles for patients and doctors to manage appointments and medical information.
  • Push Notifications: Implemented reminders for upcoming appointments and other important updates.

Technologies Used

The application was built natively for iOS using Swift and UIKit. I utilized the MVVM (Model-View-ViewModel) architectural pattern to ensure a clean and scalable codebase. For the real-time communication features, we integrated a third-party video SDK and handled RESTful API calls for data synchronization with the backend.
